## Installation

1. Add the dependency to your `shard.yml`:
```yaml
dependencies:
  IUP:
    github: dscottboggs/IUP_for_crystal
```
2. Run `shards install`

## Usage

```crystal
require "IUP"
```
See the examples directory for usage samples.
